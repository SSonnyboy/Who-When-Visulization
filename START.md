# 快速启动指南

## 方式一：使用预加载数据（推荐）

### 1. 生成合并数据文件
```bash
cd dashboard
node data-loader.js
```
这将生成 `all-data.json` 文件。

### 2. 启动本地服务器
```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx serve
```

### 3. 打开浏览器
访问 `http://localhost:8000`

---

## 方式二：直接使用（需要原始数据文件）

确保数据文件位于以下位置：
- `../data/Algorithm-Generated/*.json` (126个文件)
- `../data/Hand-Crafted/*.json` (58个文件)

然后启动本地服务器并打开浏览器。

## 故障排除

**数据无法加载？**
- 确保使用本地服务器（不要直接打开HTML文件）
- 检查数据文件路径是否正确
- 尝试运行 `node data-loader.js` 生成预加载数据

**样式或图表异常？**
- 清除浏览器缓存
- 确保网络连接正常（需要加载Chart.js）
- 查看浏览器控制台错误信息
