# 快速启动指南

## 方式一：Cloudflare Workers 本地开发（推荐）

### 1. 安装依赖
```bash
npm install
```

### 2. 启动 Worker
```bash
npm run dev
```

### 3. 打开浏览器
访问 `http://127.0.0.1:8787/who_when/`

---

## 方式二：使用预加载数据做静态预览

### 1. 生成合并数据文件
```bash
node data-loader.js
```
这将生成 `all-data.json` 文件。

### 2. 启动本地服务器
```bash
npx serve public
```

### 3. 打开浏览器
访问 `http://localhost:3000/who_when/`

---

## 故障排除

**数据无法加载？**
- 确保使用本地服务器（不要直接打开HTML文件）
- 检查数据文件路径是否正确
- 尝试运行 `node data-loader.js` 生成预加载数据

**样式或图表异常？**
- 清除浏览器缓存
- 确保网络连接正常（需要加载Chart.js）
- 查看浏览器控制台错误信息
